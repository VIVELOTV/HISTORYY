About the Files

The Stream is a mobile online TV & Video Streaming App system which run under Android platform that used for watch many online TV broadcasts & video streaming from your Android devices. You can manage unlimited TV channel and categories which support many formats, this application created by Android for client side and then PHP MySQL for server side. Run under Android platform which is the most popular operating system in the world, using this application you can save your money and time in creating application for your own online TV & Video Streaming App.

Getting Started

Before doing installation, you need to do the following things :
 
Have a web hosting to store admin panel files and Mysql database, to run this admin panel, web hosting that you use must have the following requirements :
  • PHP Version 5.4.x or Higher
  • Support Mysqli
  • Support PDO
 
Because this documentation using cpanel, a web hosting with cpanel is recommended. If you don’t have web hosting or domain, you can check on this sample site which provide hosting and domain service :
  • Sample 1
  • Sample 2
  • Sample 3

Database Creation

1. Login to your cpanel hosting and open MySQL Databases
2. Create database and give database name like you want, as example : 'solodroid_thestream_db’
3. Create user for your database
4. Add user to your database and check ALL PRIVILEGES
 
 
 
 

Import Database .sql file

1. Open phpMyAdmin and select your database
2. Import file database the_stream_db.sql.sql  from download package inside 'admin_panel' folder
 
 

Upload All Php Scripts via File Manager

1. Open File Manager via cpanel or ftp server, upload all php script inside the_stream folder to folder destination on your hosting, highly recommended to compress all files into .zip file before you upload it and extract it after the .zip file is uploaded.

2. configure the connection file, open 'config.php' in folder 'includes' which is in the includes folder of your package. That’s all, now run the webservice : http://www.yourdomain.com/the_stream/
 
3. configure your database setting :
    $host            = "localhost";
    $user            = "your_mysql_username";
    $pass           = "your_mysql_password";
    $database   = "your_database_name";

Check TV Channel Streaming URL

There are a lot of channel streaming url which available in internet, but probably there are some of them which not work properly, so, to know the channel is working or not, you can use VLC media player to check it.
 
1. Download and Install VLC media player in your computer, Click Here to Download VLC media player
2. Open VLC media player → Media → Open Network Stream.. → Select Tab Network and Enter your channel url
3. Click play, if channel can be played, most likely also can be played in android app,  otherwise it will give error dialog if channel can't be played.
